# AI Odyssey – Interactive Web Experience

AI Odyssey is an interactive storytelling website built using HTML, CSS, and JavaScript.

The project demonstrates the evolution and future of Artificial Intelligence through a scroll-based narrative experience. It includes animations using GSAP, scroll progress indicators, interactive hover cards, and a dynamic neural network background created with HTML canvas.

The website is divided into five sections: introduction, exploration of AI applications, insights into machine learning, the future of AI, and a concluding section.

Features include smooth scroll animations, responsive design, theme switching, and interactive UI components. The goal of the project is to combine storytelling and modern frontend technologies to create an engaging educational experience about Artificial Intelligence.
